# Donation-Management-System
A Fully functional Desktop Application designed on JAVA

# Tools and Language
- Netbeans
- Java
- SQL

# Description
A fully functional Desktop Application in which the functionalities are : 
- Add Donor.
- Add Beneficiery.
- Allocate Teams.
- Add Project.
- Edit Teams.
- Manage Donations.
- Manage loans.
- Maintainance.
- view details (Donor / Beneficiery)
